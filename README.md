# racegame02

C++ / SFML を用いて制作した個人ゲームプロジェクトです。
SFMLを用いた初めての製作物です。
シーン管理・状態遷移・入力処理などを自作実装しています。
こちらにYoutubeで公開しているプレイ映像のURLを貼っておきます。
https://www.youtube.com/watch?v=wcJpITbZR6Q

## 製作目的

SFMLの基本的な使い方、ゲームシーンの分割、呼び出しなどの扱いを学習するために制作しました。

## 概要
- ジャンル：レースゲーム（試作）
- 開発言語：C++
- 使用ライブラリ：SFML
- 開発環境：Visual Studio（Windows）

## 主な実装内容
- シーン管理（Title / Select / Race / Result）
- GameState / StateManager による状態遷移
- 入力処理（キーボード）
- 車両挙動のクラス分離（CarController）
- JSON を用いたデータ管理（一部）

## ビルド方法
1. Visual Studio で `racegame02.sln` を開く
2. SFML を適切に設定
3. ビルドして実行

## 備考
学習・ポートフォリオ目的で制作しているプロジェクトです。
レース開始後にシーン上の操作を表記していなかったのでこちらに表記いたします。
A,Dキー:ハンドル操作　Jキー:アクセル　Kキー:ブレーキ　
J,Kの同時押し状態では方向転換のしやすい状態になります。
